# BenefAI — AI Legal Eligibility Intelligence System

## 🚀 Overview
BenefAI is an AI-powered system that helps governments ensure welfare reaches the right people.

It detects:
- Fraudulent beneficiaries
- Duplicate applications
- Missing eligible beneficiaries

## 💡 Innovation
Unlike traditional systems that verify once, BenefAI continuously evaluates eligibility and provides explainable AI decisions.

## ⚙️ Features
- 📄 Document OCR (auto form fill)
- 🔍 AI duplicate detection
- 📊 Fraud risk scoring (0–100)
- 🕸️ Fraud ring detection (network graph)
- 🛰️ Satellite-based wealth validation (prototype)
- 🗣️ Multilingual + voice input
- 📱 WhatsApp notifications
- 📝 Explainable decisions + appeal system

## 🧠 Tech Stack
- Next.js (Frontend)
- Supabase (Database)
- Claude AI (Reasoning + OCR)
- React Force Graph (Visualization)
- Twilio (WhatsApp API)

## 🌍 Impact
Aligned with **SDG-1: No Poverty**, ensuring fair and efficient distribution of government aid.

## ⚠️ Note
This is a prototype built for hackathon demonstration using simulated data.
